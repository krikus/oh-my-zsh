@oh-my-zsh@ is an open source, community-driven framework for managing your "Zsh":http://www.zsh.org/ configuration.

It comes bundled with a ton of helpful functions, helpers, plugins, themes, and a few things that make you shout…

bq. "OH MY ZSHELL!"

h2. Setup

@oh-my-zsh@ should work with any recent release of "Zsh":http://www.zsh.org/. The minimum recommended version is *4.3.9*.

If not already installed, you can install "Zsh":http://www.zsh.org/ using the command-line.

h3. The automatic installer… do you trust me?

You can install this via the command-line with either @curl@ or @wget@.

h4. via @curl@:

@curl -L https://raw.github.com/krikus/oh-my-zsh/master/tools/install.sh | sh@

h4. via @wget@:

@wget --no-check-certificate https://raw.github.com/krikus/oh-my-zsh/master/tools/install.sh -O - | sh@


For more information visit original project: https://github.com/robbyrussell/oh-my-zsh
